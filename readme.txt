== Description ==
A plugin to simplify adding a Help Ukraine banner, from the folks at [helpukrainewin.org](https://helpukrainewin.org/), to WordPress websites.

This plugin allows you to choose from three different style variations as well as control which corner of your website the widget will display in.

== Installation ==
The widget will automatically work as soon as the plugin is installed and activated. From there, you can then go to the "Settings > Help Ukraine" page to control the display variation and position of the widget.
